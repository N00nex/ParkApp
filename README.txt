ParkFinder

Die ParkFinder App ist eine für Android basierte mobile Applikation auf der Parkplätze innerhalb einer gewünschten Reichweite zu filtern und zu sortieren, mit dem Ziel, ein passendes Parkhaus in der Umgebung zu finden.

Bevor man Loslegt

Vor dem Nutzen der App

Unter ParkApp/DBScript finden Sie den DB Script. Bitte führen Sie diesen in Ihren SSMS aus. Der Connection string ist unter ParkApp/ParkAPI/appesetings.json zu finden. Dieser ist derselbe wie beim E-Shop.

Beim Nutzen der App

Wenn die App gestartet wird, kann es passieren, dass keine Daten angezeigt werden. Um das Problem zu umgehen, machen sie folgendes:

1) Drücken Sie auf den Homebutton des Emulators (   o   )

2) Drücken Sie auf den Tab knopf und Swipe die App Weg

3) Gehen Sie Nochmals in die App

4) Jetzt müsste es funktionieren


Über den Android Emulator

Damit der Emulator die eingebaute Karte auch nutzen kann, muss man erstmals ein Standort für den Emulator bestimmen.
Das macht man folgendermassen:

1) Starte den Emulator (Android, Google Pixel 7)

2) Drücke beim Emulator Interface die drei Punkte and der Seite ( ... )

3) Navigiere zu Location an der Linken Seite

4) Füge einen Neuen Standort Hinzu

5) Bestimme den Standort (IMS Basel ist am empfehlenswerten, da die Distanzen von der Position der IMS basiert sind)
